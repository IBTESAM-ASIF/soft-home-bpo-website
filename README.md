# Soft Home BPO Website

Professional website for Soft Home BPO - Business Process Outsourcing services.

## About
This website showcases Soft Home BPO's services, case studies, and contact information for potential clients.

## Deployment
This site is built with Next.js and deployed as static files to GitHub Pages.

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

The static files will be generated in the `out/` directory.
